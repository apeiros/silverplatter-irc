color(code) click to toggle source

Foreground color with the given code. Code is 0-15. The codes translate to:

  • 0: white
  • 1: black
  • 2: blue
  • 3: green
  • 4: red
  • 5: brown
  • 6: purple
  • 7: orange
  • 8: yellow
  • 9: light green
  • 10: teal
  • 11: light cyan
  • 12: light blue
  • 13: pink
  • 14: grey
  • 15: light grey

For every color code there’s a named method too.

84:         Sequence.new(code)
85:       end
