SilverPlatter::IRC::Channel

Authors

  • Stefan Rusterholz

About

Channel represents a Channel in the IRC network.

Synopsis

Description

Channel is enumerable over the users in that channel and comparable against the normalized name.

Notes

Attributes

name[R]

The name of the channel (immutable)

topic[R]

The current topic of the channel

compare[R]

The value used for comparison/sorting

mode[R]

The modes this channel has (a String with the mode letters)

created_at[RW]

The Time this channel was created (NB: not supported by all irc servers)
