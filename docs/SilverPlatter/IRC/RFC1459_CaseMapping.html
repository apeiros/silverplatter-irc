SilverPlatter::IRC::RFC1459_CaseMapping

Authors

  • Stefan Rusterholz

About

SilverPlatter::IRC::RFC1459_CaseMapping provides the casemap method which lowercases a string according to rfc1459 (non strict)

Synopsis

  RFC1459_CaseMapping.casemap(string)
  include RFC1459_CaseMapping
  casemap(string)
